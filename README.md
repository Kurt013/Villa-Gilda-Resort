## ðŸ“¦ Installation

1. **Clone the repository**:

```bash
git clone https://github.com/Kurt013/Villa-Gilda-Resort.git
cd Villa-Gilda-Resort
```
2. **Set up XAMPP (Apache + PHP + MySQL)**:

- Start Apache and MySQL from the XAMPP control panel.

- Place the project folder inside the htdocs directory of XAMPP.

3. **Install PHP dependencies**:

- Make sure you have Composer installed, then run:

```bash
composer install
```

3. **Database setup**:

- Open phpMyAdmin at ```http://localhost/phpmyadmin``` and create a database (e.g., `villa gilda`).

- Import the provided SQL schema if available (.sql file in the repo).
